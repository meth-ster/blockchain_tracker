# blockchain_tracker
A simple TypeScript app for tracking blockchain data.

## What it does
This project provides a basic framework for monitoring and analyzing blockchain data. It allows you to track transactions, wallet balances, and other relevant information.

## Installation
To get started, clone this repository and install the dependencies:
```bash
git clone https://github.com/your-username/blockchain_tracker.git
cd blockchain_tracker
npm install
```
## Running the app
To run the app, use the following command:
```bash
npm start
```
This will start the app and make it available at `http://localhost:3000`.

## Example usage
For example, to track the balance of a specific wallet, you can use the following endpoint:
```bash
curl http://localhost:3000/wallet/balance/0x742d35Cc6634C0532925a3b844Bc454e4438f44e
```
This will return the current balance of the specified wallet in JSON format.
